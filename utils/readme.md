# Utilities

- **date_filter.py** — Filters pandas DataFrames by date range, handling partial dates (year-only, month-only) and unknown dates gracefully.
- **helpers.py** — Collection of utility functions for path resolution, YAML I/O, URL detection, document export (TXT/DOCX/PDF), and DataFrame column normalization.
- **tooltip_utils.py** — Loads tooltip strings from YAML configuration and applies them to Qt widgets by matching object names.
