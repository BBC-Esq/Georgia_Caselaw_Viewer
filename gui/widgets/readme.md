# GUI Widgets

- **chat_bubble.py** — Styled message bubble widget for chat interfaces with markdown rendering, auto-sizing, and copy-to-clipboard functionality.
- **date_filter_bar.py** — Collapsible date range filter widget with from/to date pickers and apply/clear controls.
- **search_bar.py** — Search input widget combining column selector, text field, and fuzzy matching toggle with debounced signal emission.
