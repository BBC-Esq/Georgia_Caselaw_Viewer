# GUI Dialogs

- **brief_types_dialog.py** — Dialog for creating, editing, and organizing case brief types with category management and live template preview.
- **brief_viewer.py** — Displays streaming AI-generated case briefs with export options to PDF, DOCX, and TXT formats.
- **case_chat_dialog.py** — Interactive chat interface for asking questions about a specific case, with conversation history and streaming responses.
- **preferences.yaml** — Simple configuration file storing UI preferences like font size.
- **settings_dialog.py** — Application settings dialog for configuring API keys, model selection, verbosity, and output directories with cost display.
