# Configuration Files

- **caselaw_viewer.json** — Stores user preferences including API key, model selection, verbosity settings, and directory paths.
- **caselaw_viewer_briefs.yaml** — Defines available case brief types with their labels, legal topics, and category groupings.
- **logging_config.py** — Sets up centralized logging with rotating file handlers and console output.
- **settings.py** — Manages application settings as a dataclass, handling preference loading/saving and path validation for portability.
- **tooltips.yaml** — Contains UI tooltip strings mapped to widget object names for the application's help system.
